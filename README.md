# api-platform-php
This is a base container image for api-platform, for use by the Hottomali organization.  It contains:

- A core image for use with API-platform.
- Some ancillary tags that extend that platform to include dev build dependencies.

It is primarily meant for internal use, although the public is welcome to use it as well if so desired.
